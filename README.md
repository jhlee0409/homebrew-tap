# Homebrew Tap

Custom Homebrew tap for jhlee0409 projects.

## Packages

### Claude Code History Viewer (Cask)

Browse and analyze Claude Code conversation history offline.

```bash
brew tap jhlee0409/tap
brew install --cask claude-code-history-viewer
```

To upgrade:

```bash
brew upgrade --cask claude-code-history-viewer
```

- [Claude Code History Viewer on GitHub](https://github.com/jhlee0409/claude-code-history-viewer)

### proc-janitor (Formula)

Automatic orphan process cleanup daemon for macOS/Linux.

```bash
brew install jhlee0409/tap/proc-janitor
```

Start as a service:

```bash
brew services start proc-janitor
```

- [proc-janitor on GitHub](https://github.com/jhlee0409/proc-janitor)
- [proc-janitor on crates.io](https://crates.io/crates/proc-janitor)
